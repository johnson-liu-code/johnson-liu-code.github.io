Jailbreak Stress Testing Dashboard

Interactive dashboard for stress-testing LLMs: run reproducible jailbreak suites, compare models and mitigations, and visualize success rates, leakage types, and regression trends.

Sentiment Analysis

This project explores how natural language processing can be used to recognise sarcastic intent in informal text. By training neural network models on a dataset of labelled Reddit comments, the goal is to distinguish between sarcastic and sincere statements. In future iterations the scope will expand to general sentiment analysis and even multimodal input (e.g. combining images and text).

Dominion AI

Dominion is a deck-building card game with a complex state space. This project applies reinforcement learning to train bots to play the game efficiently. Key design decisions include whether to use a fixed or dynamic action space and how to structure the Q—learning architecture for the distinct action and buy phases of the game. The work is built on top of the open—source Pyminion engine.

CivicSim

CivicSim is an interactive civic simulation game where players vote on policies and watch a simulated society evolve in response. It blends political strategy, community decision-making, and narrative world-building to explore how collective choices shape the future.

InquisitorNet

InquisitorNet is a prototype network of AI-driven bots inspired by the Warhammer 40K universe. Written in Python using the Reddit API (praw) and OpenAI's language models, each bot adopts the personality of an Imperial Inquisitor and engages with community posts. The project includes modules for logging, encryption, scheduling and database storage, providing a foundation for more elaborate behaviour in later phases.
